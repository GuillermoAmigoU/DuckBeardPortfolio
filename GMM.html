Game Maker Maker

Game Maker Maker

Game Maker Maker is a one-input Cooking Sim game where the objective is to produce assets for famous videogame franchises as fast as possible. Available on browsers and mobile (Android Store).

  • Genre: 'Cooking' simulation
  • Engine: Unity
  • Duration: 2 months
  • Objective: Creating a videogame compatible for browsers using one input.
  • Responsabilities: Game/UX Designer, Sound Designer
  • Platform: PC Browser, Mobile
  • Team Size: 6

Overview

Game Maker Maker puts you in the shoes of  an World-Wide Prop Factory employee, bringing to life props for all franchises  known. 
We all know Super Mario's mushrooms, Pokémon's Pokeballs, or Pac-Man's fruits, but where do they come from?

  • Simple Game available for mobile and Browser thanks to simplified and easy to learn controls.
  • Highly scalable for multiple franchises.
  • 8 full levels, based on different Game franchises, like Mario Bros, Pokemon, etc.
  • Fast paced and balanced/designed for highly replay-ability.
